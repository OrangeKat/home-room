# HomeRoom 🏠

A sleek, modern desktop dashboard built with **Tauri**, **Vue 3**, and **TypeScript**. Manage your life with style.

---

## 🚀 Features

- **Task Tracker**: Stay on top of your to-dos with urgency marking.
- **Habit Tracker**: Build better routines and track daily progress.
- **Dynamic Calendar**: A central view for all your activities.
- **Utility Widgets**: High-precision clock, weather updates, and quick notes.
- **Blazing Fast**: Powered by Rust and Tauri for a lightweight desktop experience.

## 🛠️ Tech Stack

- **Frontend**: [Vue.js 3](https://vuejs.org/), [Vite](https://vitejs.dev/), [TypeScript](https://www.typescriptlang.org/)
- **Backend**: [Rust](https://www.rust-lang.org/), [Tauri v2](https://tauri.app/)
- **Database**: [Supabase](https://supabase.com/)

---

## 🚦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS)
- [Rust](https://www.rust-lang.org/tools/install)
- [Supabase Account](https://supabase.com/)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/OrangeKat/home-room.git
   cd home-room
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the `src-tauri` directory:
   ```env
   SUPABASE_URL=your_supabase_project_url
   SUPABASE_KEY=your_supabase_anon_key
   ```

## 🗄️ Database Setup (Supabase)

This project uses Supabase for persistent storage. You need to create the following tables in your Supabase SQL Editor:

### 1. `tasks`

```sql
create table tasks (
  id bigint generated by default as identity primary key,
  title text not null,
  date text not null, -- Format: YYYY-MM-DD
  time text not null, -- Format: HH:MM
  is_urgent boolean default false,
  is_completed boolean default false,
  created_at timestamptz default now()
);
```

### 2. `habits`

```sql
create table habits (
  id bigint generated by default as identity primary key,
  title text not null,
  is_deleted boolean default false,
  created_at timestamptz default now()
);
```

### 3. `habit_completions`

```sql
create table habit_completions (
  habit_id bigint references habits(id) on delete cascade,
  completed_date text not null, -- Format: YYYY-MM-DD
  primary key (habit_id, completed_date)
);
```

## 🏃 Running the App

### Development Mode

To run the desktop application in development mode:

```bash
npm run tauri dev
```

If you just want to see the web frontend:

```bash
npm run dev
```

### Build for Production

To package the app for your OS:

```bash
npm run tauri build
```
